Welcome to ntop!

This page allows you to request a dump of ntop data in xml format.

WARNINGS: 1. If ntop was not configured with the --enable-xmldump option, this page will fail. Further, your request will be detected as a hostile request and your IP address will be blacklisted for a period of time.
  2. Returning a lot of data may take a LONG time, enough to exceed the timeouts on many browsers. You may want to use a non-interactive request, such as wget or lynx.
  3. If ntop was not invoked with the --xmlfilesnap= option, setting ToFile to 'yes' will fail!

Option Description Select
Version Information about ntop's configuration from version.c
Invoke Information about ntop's invokation - command line, parameter settings, etc.
Interfaces Information about each interface (NIC).
To File? Dump to a host file, not the requestor?
